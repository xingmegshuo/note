## go linux 安装
```shell
wget https://golang.google.cn/dl/go1.15.3.linux-amd64.tar.gz

tar -C /usr/local -xzf go1.4.linux-amd64.tar.gz

vim .bashrc or .zshrc

export PATH=$PATH:/usr/local/go/bin
```
